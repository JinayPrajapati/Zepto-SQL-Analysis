# Zepto Sales and Inventory Analysis using SQL

## Project Overview

This project focuses on analyzing Zepto product inventory and pricing data using SQL. The objective was to explore the dataset, clean the data, and generate business insights related to discounts, inventory, pricing, and category performance.

## Tools Used

* SQL
* MySQL Workbench

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* HAVING
* ORDER BY
* Aggregate Functions
* CASE Statements
* Data Cleaning

## Analysis Performed

* Data exploration
* Data cleaning
* Top discounted products
* Category-wise revenue analysis
* Inventory analysis
* Price-per-gram comparison
* Out-of-stock product analysis

## Key Insights

* Identified categories with the highest estimated revenue.
* Analyzed discount trends across categories.
* Detected high-value products that were out of stock.
* Compared product pricing efficiency using price-per-gram metrics.
